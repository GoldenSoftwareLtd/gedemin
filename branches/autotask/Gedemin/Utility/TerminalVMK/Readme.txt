Инструкция по  проекту “Терминал”.
Проект написан для 2-х терминалов: SKORPIO X3 и SKORPIO.
Скачиваем исходники.
Настройка IDE:
1.	Проект – параметры проекта – параметры компилятора - режим сборки – wince
2.	Проект – параметры проекта – параметры компилятора - генерация кода – операционная система – wince
3.	Проект – параметры проекта – параметры компилятора - генерация кода – целевое семейство процессоров – arm
4.	Для SKORPIO X3: 
            Проект – параметры проекта - параметры компилятора – дригие -  “-dSKORPIOX3” 
            Для SKORPIO не должно быть  “-dSKORPIOX3” .     
Откомпилированному файлу(.exe) для модели  SKORPIO нужна DLL “scanner_dll”, которая должна находится возле исполняемого файла.
